Arreglos 💅🏽

Los arreglos se declaran con corchetes y cada elemento es considerado una posición del arreglo.

Su sintaxis básica es la siguiente:
let arreglo = [pos0, pos1, pos2, ..., posN]

Este es un arreglo vacío:
let arregloVacio = []; (corchetes sin nada dentro)

Para saber el tamaño del arreglo se puede usar arreglo.length
donde la posición tiene el siguiente conteo:
let colores = ['negro', 'rojo', 'rosa', 'verde']
let colores = [0, 1, 2, 3]
